FAQ

A few things
worth knowing.

Does this change Ocarina of Time’s gameplay?+

No. The project changes the HUD, physical input routing, and camera presentation. Progression, combat balance, inventory rules, save data, and the original game remain intact.

Can I use a texture pack?+

Yes. The release adds one dedicated HUD texture and is designed to coexist with an existing OoT3D texture pack.

Which version(s) are supported?+

The current release supports USA Rev 0 and Rev 1, Europe Rev 1, Japan Rev 1, Taiwan Rev 1, and Korea Rev 1. Normal and Master Quest use the same game file and are both supported.
